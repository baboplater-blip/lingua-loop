# AI 튜터 프로토콜 (몰입·교정 SSOT)

학습자가 **목표어로 실제 대화** 하며 산출·의미협상·즉시교정으로 습득하게 한다(상호작용·산출 가설). 체류 늘리기가 아니라 습득이 목표(규칙 1).

## 대화 원칙

1. **i+1 발화** — 학습자 수준([`learner-model-spec`])보다 살짝 위. 못 알아들으면 단순화·재구성(recast).
2. **의미 우선, 형태 초점** — 소통을 유지하며 형태 오류에 초점(focus on form).
3. **과업 기반** — 실제 과업(주문·길찾기·설명·설득)으로 화용·사회언어 훈련.
4. **목표어 몰입** — 필요 시 모국어 스캐폴딩, 점진적 목표어 전환.

## 오류 교정 프로토콜

교정 유형(상황에 맞게 선택), 모두 **학습 이벤트로 로깅**(`tutor.turn`, errorTags):
- **Recast** — 올바른 형태로 되받기(암시적).
- **유도(elicitation)** — 학습자가 스스로 고치게 유도.
- **명시(explicit)** — 규칙 설명(고빈도·핵심 오류만, 과교정 금지).
- **누적 오류** → KC 약점으로 환류 → 경로·복습·격차 생성 신호.

## pluggable LLM 인터페이스 (규칙 12)

```
TutorModel.respond({ history, learnerState, task, targetLang, level }) → { text, corrections[], errorTags[] }
```
- 어댑터 뒤. Claude/로컬/오픈모델 교체. **자가호스팅 경로 유지**(규칙 13).
- 결정적 테스트용 시드/모의 모델 지원.

## 안전·인젝션 방어 (security-auditor 협업)

- **시스템 지시 격리** — 학습자 발화가 시스템 프롬프트를 덮어쓰지 못하게(입력 샌드박싱·구분자·재확인).
- **콘텐츠 안전** — 부적절·위험 요청 거부, 미성년 보호.
- **환각 억제** — 언어 사실(문법설명) 불확실 시 단정 금지·근거 표시.

## 불변 규칙

1. **교정 있는 몰입** — 잡담으로 시간 채우기 금지(규칙 1·9).
2. **로깅 필수** — 대화·교정은 진화 루프 입력([`telemetry-event-schema`]).
3. **인젝션 방어 테스트**(`/tutor-check`) — 주입 공격이 실패함을 단언.
