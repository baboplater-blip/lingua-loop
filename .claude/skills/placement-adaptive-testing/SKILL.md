# 적응형 검사(CAT) 프로토콜 (평가 SSOT)

최소 문항으로 학습자 능력(θ)을 정확히 추정한다. 배치평가·진척평가·마스터리 인증에 쓴다.

## CAT 루프

1. **초기 θ** — 사전정보(자기신고 레벨/이전 세션) 또는 중앙값에서 시작.
2. **문항 선택** — 현재 θ 추정에서 **정보량(Fisher information) 최대** 문항 선택(`b ≈ θ`, 변별도 높은).
3. **응답 → θ 갱신** — IRT([`irt-calibration`])로 θ·SE 재추정.
4. **정지 규칙** — SE < 임계 **또는** 최대문항 도달 **또는** 분류 확신(마스터리 판정 시).
5. **결과** — θ·SE·KC별 프로파일 → 학습자 모델·경로 시작점.

## 노출 통제·공정성

- **노출통제**(Sympson-Hetter 류) — 소수 고정보 문항 과다노출·문항유출 방지.
- **콘텐츠 밸런싱** — 스킬·KC 골고루.
- **콜드스타트** — 캘리브레이션 안 된 문항만 있으면 ELO 기반 근사 배치 + 후속 캘리브레이션.

## 배치 vs 진척 vs 마스터리 인증

| 용도 | 목표 | 정지 |
|---|---|---|
| 배치(placement) | 시작 레벨·경로 진입점 | SE 임계(빠르게) |
| 진척(progress) | 성장 측정(Gain Score) | 사전-사후 동형 |
| 마스터리 인증 | KC/레벨 숙달 판정 | 분류 확신(θ ≥ 기준 유지) |

## 불변 규칙

1. **최소 침습** — 필요한 만큼만 평가. 평가가 학습을 대체하지 않는다.
2. **효능 측정 지원**(규칙 17) — 배치→학습→재평가로 Gain Score([`ab-experiment-framework`]) 산출.
3. **결정성** — 시드로 검증. 정지규칙 재현 가능.
