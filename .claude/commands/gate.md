# /gate

[test.md](../../test.md) 의 완료 정의를 집행하는 **단일 판정자**([`verification-gate`](../skills/verification-gate/SKILL.md)). 그린 아니면 "완료"도 배포도 없다(규칙 15).

## 절차 (`qa-verification-engineer` 오케스트레이션)

1. **빌드 + 클린 스테이트** — 테스트 데이터 초기화(누적 cruft 레이스 방지).
2. **전 카테고리 검증**(test.md §3) — 콘텐츠·SRS·심리측정·텔레메트리·진화·튜터·발음·**효능**·프라이버시·커뮤니티·다국어·문서.
3. **불변식 = 실패조건** — append-only 위반·PII 누출·미검증 콘텐츠 노출·라이선스 누락 시 FAIL.
4. **다크패턴 안티테스트**(규칙 9) — 손실공포·강제알림·죄책감 카피 **없음** 단언.
5. **프라이버시 레드팀** — 개방셋 재식별 시도 실패 단언.
6. **효능 스모크**(test.md §6) — 시뮬 학습자로 스케줄러 리텐션 개선·캘리브레이션 수렴·불량문항 차단.
7. **문서 자기검증** — 5문서 존재·정합·상호링크(규칙 22).

## 출력 형식

```
✓ /gate
  콘텐츠 ✅ SRS ✅ 심리측정 ✅ 텔레메트리 ✅ 진화 ✅ 튜터 ✅ 발음 ✅
  효능 ✅ 프라이버시 ✅ 커뮤니티 ✅ 다국어 ✅ 문서 ✅
  다크패턴 안티테스트 ✅ / 재식별 레드팀 ✅
  → ✅ 게이트 통과
```

## 주의

- `--quick` 은 API/유닛만(브라우저 E2E 생략).
- 확률요소는 시드 고정으로 결정화(플레이크 금지). 우회 통과 금지.
