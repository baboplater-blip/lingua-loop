# 모델 카드 (Model Card)

> 규칙 14·17: **카드 없이 모델 공개 금지.** LinguaLoop의 "모델"은 거대 신경망이 아니라, 학습 데이터로 **적합·재적합되는 학습과학 알고리즘의 묶음**입니다. 각각의 용도·학습 데이터·한계·편향·평가·pluggable 대안을 밝힙니다.

## 1. 구성 모델

| 모델 | 역할 | 적합 데이터 | 위치 |
|---|---|---|---|
| **FSRS 스케줄러** | 복습 간격(기억 안정성·난이도·인출확률) | 개인 복습 로그 | `core/fsrs.ts` (파라미터화, `engine/fsrs-optimize`가 재적합) |
| **IRT/ELO 캘리브레이션** | 문항 난이도·변별도 추정 | 응답 정오 로그 | `core/calibration.ts` |
| **BKT 숙달 추적** | 지식요소별 숙달 확률 | 응답 로그(리플레이) | `core/learner-model.ts` |
| **CAT 배치** | 적응형 능력추정·배치 | 응답 로그 | `core/placement.ts` |
| **로컬 튜터**(en·es·기타) | 대화·오류 교정·i+1 (en: 관사·be·3인칭 / es: ser·성 일치 / 그 외: 패스스루=오교정 방지) | 규칙 기반(무학습) | `adapters/local-tutor.ts`·`spanish-tutor.ts`·`tutor.ts` |
| **콘텐츠 생성기**(en·es) | 격차 KC 문항·읽기 지문 생성 | 템플릿·렉시콘(무학습) | `adapters/content-gen.ts`·`spanish-content.ts`·`reading-gen.ts` |
| **발음 스코어러**(`LocalPhoneticScorer`) | IPA 자질거리 정렬(분절음) + **운율(강세)** 채점 | IPA 조음 자질표·강세 패턴(무학습) | `core/phonetics.ts`·`adapters/pronunciation.ts` |
| **학습효과 측정**(`itemEffects`) | 문항 실효(난이도 적합+변별도)로 커뮤니티 콘텐츠 재랭킹·강등 | 사용 응답 로그(ELO 능력추정) | `core/content-effect.ts`·`engine/community-effect.ts` |
| **검토자 신뢰가중**(`computeTrust`) | 검토 표를 이력 신뢰도로 가중(안티어뷰즈) | 검토-객관진실(게이트·효과) 일치 이력 | `core/community.ts` |

## 2. 의도된 용도

- 개인화된 **마스터리 학습**: 무엇을·언제 복습하고, 다음에 무엇을 배우며, 산출을 어떻게 교정할지 결정.
- **진화 루프의 구성요소**: 수집 데이터로 위 파라미터를 재적합해 교수 효율(TTM 단축·리텐션↑)을 개선(`/evolve`).

## 3. 하지 말아야 할 용도 (Out-of-scope)

- 학습자 순위 매기기·낙인·고부담 시험 판정에 단독 사용 금지.
- 참여도(체류·스트릭) 극대화 목적의 최적화 금지 — **가드레일이 학습성과이므로**(규칙 1), 성과를 해치며 참여만 늘리는 변경은 알고리즘 레벨에서 반려됩니다.

## 4. 한계·편향

- **콜드 스타트**: 데이터가 적은 초기에는 기본 파라미터(사전분포)에 의존해 추정 불확실성이 큽니다.
- **언어 자원 편향**: 시드 콘텐츠가 풍부한 언어(en)가 소수 언어보다 캘리브레이션 품질이 높습니다. 저자원 언어 우대는 커뮤니티·거버넌스로 보완합니다.
- **로컬 튜터의 범위**: 기본 튜터는 결정적 규칙 기반이라 다룰 수 있는 오류 유형이 제한적입니다(관사·be 일치·3인칭 -s·대문자 등). 더 넓은 교정은 pluggable LLM 어댑터로 교체합니다.
- **생성 콘텐츠**: 템플릿 생성물은 다양성이 제한적이며, **품질 게이트를 통과한 것만** 노출됩니다(규칙 4).

## 5. 평가 (효능)

`npm run gate`의 효능 스모크가 "동작"이 아니라 "성과에 이로움"을 검증합니다(규칙 7):

- FSRS 스케줄 > 고정 간격(리텐션/효율)
- IRT 난이도 회수 상관 r > 0.9
- CAT 능력추정 수렴
- 품질 게이트가 불량·중복 문항 차단
- **진화 자기개선**: 재적합 후 held-out 예측오차 감소(과적합 아님)
- **가드레일**: 복습만 늘리는(성과 동일) 변경은 반려

## 6. Pluggable 대안 (규칙 12·13)

튜터·콘텐츠 생성기는 인터페이스 뒤에 있어 외부 LLM으로 교체 가능하지만, **기본값은 오프라인 로컬**이라 외부 API 없이도 완전히 동작합니다(자가호스팅). 외부 모델을 붙일 때는 `withSafety` 인젝션 방어를 유지하고, 그 모델의 라이선스·편향을 이 카드에 추가로 명시해야 합니다.

- 외부 LLM 어댑터를 붙일 경우 **그 모델의 별도 라이선스·용도 제한**을 이 카드에 추가 명시해야 합니다.
